자바스크립트는 동적인 웹사이트 니즈에 맞춰서 만들어진 스크립트 언어

네스케이프에서 만듬, 이때 마소가 배껴서 jscript라는 언어를 만들고 IE를 출시하고 점유율을 높임

브라우저 마다 쓰는 스크립트 언어가 달라서 네스케이프에서는 ECMA 단체를 찾아가서 표준화를 하자고함

이때 처음 ECMAScript v1이 나옴

우리가 지금 표준으로 쓰고 있는 ECMA v5, v6의 시작임

그 후 회사간의 알력 다툼으로 표준이 잘 안잡히다가 크롬이 나온 이후 다시 표준화를 시작함

지금은 엄청 발전해서 SPA플랫폼으로 사용되고 노드로 서버 모듈도 작성가능하고

React Native, Cordova등을 이용하여 모바일 개발도 가능해짐

요즘은 Web Assebly가 나오면서 기존의 서버 언어인 C, JAVA, python등으로 웹 개발이 가능해짐
